## INSTALLATION

1. Run command `npm install -S ui-kit`
2. Export style

   <pre>
   <code>
   import 'ui-kit/lib/style.css'
   </code>
   </pre>

3. Export theme

   <pre>
   <code>
   import 'ui-kit/lib/themes/light.css'
   </code>
   </pre>
   or
   <pre>
   <code>
   import 'ui-kit/lib/themes/dark.css'
   </code>
   </pre>

3. Use awesome components
